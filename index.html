计算机组成原理

Posted on 2024-06-15 Edited on 2024-10-08

计算机系统概述

  1. 冯·诺依曼机的特点

    • 计算机由五大部件组成：运算器、控制器、存储器、输入设备、输出设备
    • 指令和数据以同等地位存于存储器，可按地址寻访
    • 指令和数据用二进制表示
    • 指令由操作码和地址码组成
    • 存储程序
    • 以运算器为中心（现代计算机以存储器为中心）

  2. \(1\) 字节(Byte) = \(8\) 比特(bit) 即 \(1B\) = \(8b\)

  3. 在存储容量中：\(1K = 2^{10}, 1M = 2^{20}, 1G = 2^{30}, 1T = 2^{40}\)
    与时间相关中：\(1K = 10^3, 1M = 10^6, 1G = 10^9, 1T = 10^{12}\)
    \(1s = 10^3ms, 1ms = 10^3us, 1us = 10^3ns, 1ns = 10^3ps\)
